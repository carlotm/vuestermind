## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
NODE_OPTIONS=--openssl-legacy-provider npm run serve
```

And then visit [http://localhost:8080](http://localhost:8080)

### Run unit tests
```
NODE_OPTIONS=--openssl-legacy-provider npm run test:unit
```
